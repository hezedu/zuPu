zuPu
====

JS powfull dump!

npm install zupu.js;
用法:<br/>
前端(browser)：
只需:
< script src="....zuPu.js " ></ script >
(window): document.write(zuPu( var )) .
注意:前端是大写!


后端(nodejs): 
<div>var zuPu=require('zuPu.js').F .</div>
记住一定加 .F 哦 . MUST BE .F  .<br/>
还有再加上前端不然没样式:
< script src="....zuPu.js"> < / script><br/>
然后 
res.write(zuPu( var )) .
